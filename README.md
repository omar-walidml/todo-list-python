#TODO LIST APP

A simple Python project to manage your daily tasks efficiently. Add tasks, mark them as complete, and delete them easily—all from the command line. Tasks are saved using JSON.

Features
	•	✅ Add new tasks
	•	✅ View all tasks
	•	✅ Mark tasks as completed
	•	✅ Delete tasks
	•	✅ Persistent storage with JSON

How to Run
	1.	Clone the repository
  2.  Navigate to the project folder
  3.  Run the app
  4.	Follow the on-screen instructions to manage your tasks
